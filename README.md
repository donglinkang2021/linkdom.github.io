# README


## 项目目的

这个项目的目的是想在手机端和笔记本上迅速同步，方便自己的日常记录；

## 项目来源

今晚是2022.9.15号，由于一堆事情接踵而至，自己在笔记本写的todo list也没能及时更新，
于是想着通过github来暂时顶替一下；

今天是2022.9.17号，今天是周末，刚好有一个早上是完全有时间的，于是查了别人的博客来照猫画虎，
主要技术是用了mkdocs，然后用github pages来托管，这样就可以通过github.io来访问了；
